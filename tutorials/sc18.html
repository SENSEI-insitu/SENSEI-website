Supercomputing 2018

In 2018, we presented a half-day tutorial on SENSEI on Sunday, November 11th. See the SC18 program for details. The course materials are available here:

  • Slides (PDF, 57 MB)
  • Virtual machine disk image (OVA, 8.9GB)

The disk image is based on Ubuntu 18.04 and can be used with VirtualBox to try SENSEI out without having to build it yourself. However, once you decide to use it with your simulation, you should build it yourself to avoid the overhead of a virtual machine.

For those who attended the tutorial at Supercomputing, we also had temporary accounts and a small reservation set up on Cori @ NERSC, a real supercomputer, that you could use to try out SENSEI during the tutorial! We have found this a useful way to provide access to SENSEI without large downloads.
